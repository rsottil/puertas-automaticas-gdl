## Deploy en GitHub Pages

### Opción 1: GitHub Pages (Gratis)

1. Crear repositorio en GitHub
2. Subir `index.html`
3. Settings → Pages → Source: "Deploy from branch" → Branch: `main` → Folder: `/ (root)`
4. Tu sitio estará en: `https://tu-usuario.github.io/nombre-repo`

### Opción 2: Cloudflare Pages (Recomendado)

1. Conectar repo de GitHub a Cloudflare Pages
2. Build command: (dejar vacío)
3. Output directory: `/`
4. Agregar dominio personalizado
